## 🚦 Como rodar o projeto

1. **Clone o repositório:**
   ```sh
   git clone <repository-url>
   cd portifolio-react
   ```

2. **Instale as dependências:**
   ```sh
   pnpm install
   # ou npm install
   # ou yarn install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```sh
   pnpm dev
   # ou npm run dev
   # ou yarn dev
   ```

4. **Acesse:**
   [http://localhost:3000](http://localhost:3000)
